[CQOI2018]异或序列

字数统计: 507字 | 阅读时长: 2分

传送门

题意：

已知一个长度为n的整数数列$a_1,a_2,…,a_n$,给定查询参数l、r,问在$a_l,a_{l+1},…,a_r$区间内，有多少子序列满足异或和等于k。也就是说，对于所有的$x,y (I ≤ x ≤ y ≤ r)$,能够满足$a_x \bigoplus a_{x+1} \bigoplus … \bigoplus a_y = k$的$x,y$有多少组。

数据范围：$1≤n,m≤10^5,0≤k,a_i≤10^5,1≤l_j≤r_j≤n$
